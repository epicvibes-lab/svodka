# Сводка — личная лента важных новостей

Утренняя сводка мировых и политических новостей на русском. Сборщик по расписанию
читает RSS, отбирает главное через Claude (модель Haiku) и пишет `svodka.json`.
Дашборд `svodka.html` просто показывает готовую сводку — открывается с любого устройства.

Инфраструктура бесплатна (GitHub). Платишь только за вызовы модели — центы в месяц.

## Файлы

- `svodka.html` — дашборд (то, что ты открываешь).
- `build.mjs` — сборщик: RSS → дедупликация → отбор через Claude → `svodka.json`.
- `package.json` — одна зависимость (`rss-parser`).
- `.github/workflows/update.yml` — расписание (cron) и автокоммит сводки.
- `svodka.json` — появится после первой сборки.

## Настройка (один раз)

1. **Создай репозиторий на GitHub** (рекомендуется Public — безлимитные минуты
   Actions и бесплатный хостинг Pages) и загрузи в него все эти файлы, сохранив структуру
   (`update.yml` должен лежать именно в `.github/workflows/`).

2. **Получи API-ключ Anthropic.** Заведи аккаунт на платформе Claude, пополни баланс
   (хватит $5), выпусти ключ. Постоянного бесплатного тарифа у API нет.

3. **Добавь ключ в секреты репозитория:**
   Settings → Secrets and variables → Actions → New repository secret.
   Имя: `ANTHROPIC_API_KEY`, значение — твой ключ.

4. **Включи GitHub Pages:**
   Settings → Pages → Build and deployment → Deploy from a branch → ветка `main`, папка `/ (root)`.
   Через минуту страница будет доступна по адресу `https://<твой-логин>.github.io/<репозиторий>/svodka.html`.

5. **Запусти первую сборку вручную:**
   вкладка Actions → workflow «Update svodka» → Run workflow.
   Появится `svodka.json`, и дашборд оживёт. Дальше сборка идёт сама по расписанию.

## Как менять

- **Частота обновления** — строка `cron` в `.github/workflows/update.yml`.
  Раз в день: `'0 5 * * *'`. Каждые 6 часов: `'0 */6 * * *'`. Время в UTC.
- **Источники** — массив `FEEDS` в `build.mjs`. Можно добавлять любые RSS-ленты.
- **Сколько новостей и какая модель** — константы `PICK` и `MODEL` в начале `build.mjs`.

## Стоимость

GitHub — бесплатно. Один прогон сборки на модели Haiku ≈ $0.007.
Раз в день ≈ 20 центов в месяц; каждый час ≈ ~$5 в месяц.
Веб-поиск модели тут не используется (новости даёт RSS), так что лишних трат нет.
